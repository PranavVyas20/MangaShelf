Assignment project for Marrow. 
Made using:
- [Kotlin]
- [Jetpack Compose]
- [Room Database]
- [Flows]
- [Retrofit]

Features:
- [Caches data locally for no internet access]
- [Loads data lazily through paging]
- [Sync lazy list with tab row]
- [Filter mangas by score, year published, popularity]
- [Bookmark a manga]
